## Release Highlights

- More Compact components landed, making complex composition patterns more consistent.
- Date and time related component coverage expanded further across input and selection flows.
- Multiple component and shared-layer optimizations shipped, and the docs surface itself continued to improve.

## Upgrade Suggestions

1. Start from this page, then open the full changelog against the components you actually use.
2. If the upgrade touches date, calendar, navigation, or table workflows, revisit those component docs first.
3. After the upgrade, use the component catalog to spot any newer component or Compact pattern that can simplify your current implementation.
