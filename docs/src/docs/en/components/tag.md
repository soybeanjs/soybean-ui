# Tag

## Overview

A small, interactive component used to categorize or label content.

## Usage

```vue
<script setup lang="ts">
import { STag } from '@soybeanjs/ui';
</script>

<template>
  <STag>Default</STag>
  <STag color="primary" variant="soft">Primary</STag>
</template>
```
