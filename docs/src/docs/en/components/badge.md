# Badge

## Overview

A small, interactive component used to categorize or label content.

## Usage

```vue
<script setup lang="ts">
import { SBadge } from '@soybeanjs/ui';
</script>

<template>
  <SBadge>Default</SBadge>
  <SBadge color="primary" variant="soft">Primary</SBadge>
</template>
```
