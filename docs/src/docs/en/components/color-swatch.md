# Color Swatch

## Overview

A read-only color preview block with support for transparent and OKLCH values.

## Usage

```vue
<script setup lang="ts">
import { SColorSwatch } from '@soybeanjs/ui';
</script>

<template>
  <SColorSwatch color="oklch(62% 0.22 312)" />
</template>
```
