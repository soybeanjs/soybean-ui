# Skeleton

## Overview

A skeleton placeholder component for loading states.

## Usage

```vue
<script setup lang="ts">
import { SSkeleton } from '@soybeanjs/ui';
</script>

<template>
  <SSkeleton class="h-4 w-56" />
</template>
```
