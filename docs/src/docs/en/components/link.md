# Link

## Overview

A styled anchor tag or router link component.

## Usage

```vue
<script setup lang="ts">
import { SLink } from '@soybeanjs/ui';
</script>

<template>
  <SLink href="https://github.com" target="_blank">External Link</SLink>
  <SLink to="/home">Internal Link</SLink>
</template>
```
