# 徽章

## 概述

一个小巧的交互组件，用于对内容进行分类或标记。

## 用法

```vue
<script setup lang="ts">
import { SBadge } from '@soybeanjs/ui';
</script>

<template>
  <SBadge>默认</SBadge>
  <SBadge color="primary" variant="soft">Primary</SBadge>
</template>
```
