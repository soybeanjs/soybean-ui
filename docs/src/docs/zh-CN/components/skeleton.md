# 骨架屏

## 概述

用于在内容加载期间展示占位结构的骨架屏组件。

## 用法

```vue
<script setup lang="ts">
import { SSkeleton } from '@soybeanjs/ui';
</script>

<template>
  <SSkeleton class="h-4 w-56" />
</template>
```
