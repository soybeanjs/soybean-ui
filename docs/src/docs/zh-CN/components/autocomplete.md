# 自动完成

## 概述

用于根据输入内容筛选建议项并快速填充文本的自动完成组件。

## 用法

```usage
basic
```

> `SAutocomplete` 现在会把筛选与选项聚合逻辑委托给 headless `AutocompleteCompact`。如果需要无样式、数据驱动的组合入口，可从 `@soybeanjs/headless/autocomplete` 直接导入 `AutocompleteCompact`。
