# 表格

## 概述

用于展示行列数据的数据表格组件。支持分组表头、排序、筛选、选择、展开、树形行、虚拟滚动和空状态展示等功能，并提供 default 与 simple 两种视觉变体，以及 rounded 圆角开关。
