# 标签页

## 概述

用于在多个内容面板之间切换的标签页组件。

## 用法

```usage
basic
```

> `STabs` 现在会把条目遍历、默认 trigger/content 组合与 indicator 渲染委托给 headless `TabsCompact`。如果需要无样式、数据驱动的组合入口，可从 `@soybeanjs/headless/tabs` 直接导入 `TabsCompact`。
