# 选择器

## 概述

用于从选项列表中选择值的选择器组件。

## 用法

```usage
basic
```

> `SSelect` 现在会把选项聚合与默认条目组合逻辑委托给 headless `SelectCompact`。如果需要无样式、数据驱动的组合入口，可从 `@soybeanjs/headless/select` 直接导入 `SelectCompact`。

## 演示

```playground
basic
default-value
disabled
separator
group
multiple
```
