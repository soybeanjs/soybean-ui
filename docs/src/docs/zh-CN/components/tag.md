# 标签

## 概述

用于标记、分类或展示状态的标签组件。

## 用法

```vue
<script setup lang="ts">
import { STag } from '@soybeanjs/ui';
</script>

<template>
  <STag>默认</STag>
  <STag color="primary" variant="soft">Primary</STag>
</template>
```
