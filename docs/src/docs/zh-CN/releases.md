# Releases

## 概述

这里集中展示 Soybean UI 的版本演进、最近一次发布内容与升级入口，适合在开始升级前先快速了解本次版本重点。

## 最新发布

| 项目         | 内容                                                                                                                                               |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| 版本         | `v0.16.0`                                                                                                                                          |
| 日期         | `2026-05-01`                                                                                                                                       |
| 重点         | Card / Popover / Editable / HoverCard / NavigationMenu / Pagination / Stepper 的 Compact 能力补齐；新增 calendar、date、month、time、year 系列组件 |
| 查看完整变更 | [GitHub Changelog](https://github.com/soybeanjs/soybean-ui/blob/main/CHANGELOG.md)                                                                 |

## 本次版本亮点

- 补齐多组 Compact 组件，进一步统一复杂组件的组合方式。
- 时间与日期相关组件体系继续扩展，覆盖更完整的录入与选择场景。
- 多个组件与 shared 能力完成优化和修正，文档页本身也继续重构。

## 升级建议

1. 先看本页，再打开完整 changelog 对照你实际使用到的组件。
2. 如果升级涉及日期、日历、导航或表格相关能力，优先回看对应组件页面的演示和 API。
3. 完成升级后，再用组件目录页快速检查是否有更适合的新组件或 Compact 结构可以替换现有实现。
