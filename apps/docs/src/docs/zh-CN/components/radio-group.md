# 单选框组

## 概述

用于在多个选项中选择单个值的单选框组组件。

## 用法

<UsageCode component="radio-group" />

> `SRadioGroup` 现在会把条目遍历和默认单选项结构组合委托给 headless `RadioGroupCompact`。如果需要无样式、数据驱动的组合入口，可从 `@soybeanjs/headless/radio-group` 直接导入 `RadioGroupCompact`。
